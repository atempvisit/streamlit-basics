# This repository shows the basics of python's streamlit framework.

Required packages:
- streamlit

To install the required packages, run the following command in your terminal:
```
pip install streamlit
```

To run the streamlit app, use the following command in your terminal:
```
streamlit run [filename].py
```

Replace `[filename].py` with the name of your streamlit app file. This will start the streamlit server and open the app in your default web browser.

In the app.py a gemini api usage is shown.

To use the gemini api, you need to install the google-generativeai package. You can do this by running the following command in your terminal:
```bash
pip install google-genai
```
Make sure to replace the placeholder values in the code with your actual API key and model name. You can obtain an API key from the Google Cloud Console and choose a model that suits your needs.
